Setup Git

Setup Github pages with custom domain

  1. Create a repository names as $(your_github_account_name).github.io. Remember $(your_github_account_name) must be exactly your github account name. This would be your GitHub Pages site’s repository, whose index page will be shown on
    http://$(your_github_account_name).github.io.
  2. Setup the GitHub Pages repository, so that your Github Pages can be redirected to your custom domain.
  3. Setup your domain provider, so that your Github Pages can be linked together with your custom domain.
