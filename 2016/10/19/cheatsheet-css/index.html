CSS Cheatsheet

Posted on 2016-10-19   |   In Tech   |  

WARNING: VERY CHAOTIC
This is just a CSS reminder for my self ;)

  • box-sizing property
    Whether width and height properties (and min/max properties) includes only the content(content-box) or also include border & padding(border-box). Applied to elements with width & height.
  • initial keyword
    Set a CSS property to its default value. Can be used for any CSS property, and on any HTML element.
  • inherit keyword
    Specifies that a property should inherit its value from its parent element. Can be used for any CSS property, and on any HTML element.
  • Set up element’s background
    • background-image: url(...)
      Apply to body element to set the background image of the page.
    • background-size: contain
      auto & initial(image’s real size); <length> & <percentage>(px or percent of the parent element; for percentage, if only one value is given, the second one would be set as auto); cover(image as large as possible even there’s an overflow); contain(image as large as possible but stretch within the content area). EXAMPLE
    • background-position: 50% 50%;
      Sets the starting position(in percentage or in px) of a background image. Default: bg-image is placed at the top-left corner of an element, and repeated both vertically and horizontally.
    • background-repeat: no-repeat;
      Repeat or not in x/y axis.

  • Set image within a box:
    To make the image fill the box, remember to set width: 100%; height: 100%; Example:

  • position property

    • static: [default value] the element won’t be positioned(will stay at its default position).
    • relative: the element will be positioned relative to its default position(AKA its static position).
    • fixed: the element will be positioned fixed relative to the viewport.
    • absolute: the element will be positioned fixed relative to its nearest positioned (any position property except static) parent.
  • display property
    block / inline / none(vs visibility:hidden) / flexbox

  • CSS animation
    Refer to The CSS Animations Pocket Guide. You can only read the first chapter to start with the most basic things, and use the rest of the book as a reference for all the other properties.

  • CSS3 Transitions
    Change property values smoothly (from one value to another), over a given duration.

  • CSS3 gradients
    • 2 types: Linear Gradients (goes down/up/left/right/diagonally)
      Radial Gradients (defined by their center)
    • Syntax:
      • background: linear-gradient(direction, color-stop1, color-stop2, ...);
      • background: linear-gradient(left, red, yellow);
      • background: linear-gradient(135deg, rgba(232,232,232,1) 0%, rgba(254,254,254,1) 100%);
  • CSS3 shadow
    • 2 types: text-shadow and box-shadow
    • Syntax: text-shadow/box-shadow: 1px 1px 2px -4px black, 0 0 25px blue;
      Means: this text/box contains 2 shadows; shadow1 is in black color, its offset-x is 1px, offset-y is 1px, blur-radius is 2px, spread-radius is -4px (which means the shadow is 4px smaller than original element); shadow2 is blue, 0px offest-x, 0px offset-y, 25px blur-radius.
  • transform property: transform: scale($ratio)
  • overflow property: hidden means to hide the son element’s overflow outside of the parent area.
  • @media rule
    Define different style rules for different media types/devices (such as viewport width/height; device width/height in different
    orientations (is the tablet/phone in landscape or portrait mode?);
    resolution, etc.)
  • display: flex
    More flexible containers for layout. Supported by limited browsers.
  • justify-content: center
    In the target flex container, position the inside items at the center of the container. This property is used to align the flexible container’s items when the items do not use all available space on the main-axis (horizontally).
  • align-items: center
    Similar to justify-content, but is used to align the items vertically.
  • text-transform: uppercase
    Controls the capitalization of text.
  • font-size: 1.4rem
    • “Root em”
    • rem unit vs em unit: em is calculated relative to its parent, while rem is calculated from the root font size. If we set ul { font-size: 0.75em }, ul font-size would be 16px*0.75, but nested ul ul font-size would be 16px0.750.75. If we set ul { font-size: 0.75rem }, both ul and ul ul font-size would be 16px*0.75.
  • &:hover, &:focus
    A syntax in SASS and LESS. & will be replaced with the parent selector as it appears in the CSS. Example here.
  • Psuedo elements
    • :before (or ::before) is treated as the first child of the element it is being applied to, whilst :after (or ::after) is treated as the last child. Thus when setting up their position and size, just consider them to be the child elements of the applied element.
    • are inline boxes
  • width
    • Percent is a measurement unit relative to the containing block.
    • inherit means inheriting this property from its parent element.
  • text-indent: 50px
    Indent the first line inside the target element with 50 pixels.
  • outline: green dotted thick
    Specifies the style of the element’s outline. Difference between border and outline: The CSS border properties allow you to specify the style and color of an element’s border. An outline is a line that is drawn around elements (outside the borders) to make the element “stand out”. The outline shorthand property sets all the outline properties in one declaration.
  • vertical-align: middle
    To align text and elements next to text. Works on inline elements(display should be inline/inline-block/inline-table). Take a look at this blog.

  • text-align
    Specifies the horizontal alignment of text in an element.

  • float

Layout & Position Practices:

  • 2: box-shadow; background linear-gradient; overflow hidden; width 100% to take up the whole container; transform scale(ratio)
